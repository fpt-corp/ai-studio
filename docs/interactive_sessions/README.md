# Interactive Sessions

---

## Quick Start

After signing in to the FPT AI Factory Portal at [https://ai.fptcloud.com/](https://ai.fptcloud.com/) or [https://ai.fptcloud.jp/](https://ai.fptcloud.jp/), follow the instruction below:

---

### Step 1: Add credit to account

* Select **Billing** in the **Sidemenu**.
* Click **Add Credit** button and enter an amount and payment information to complete.
   Or, if you have a voucher from FPT, apply your valid code in the **Voucher code** to redeem credits.
* Follow the detailed guide here:
   [https://fptcloud.com/en/documents/ai-factory-billing/?doc=billing-calculation](https://fptcloud.com/en/documents/ai-factory-billing/?doc=billing-calculation)

---

### Step 2: Create session

* Select **Model Testing** in the **Sidemenu**, and navigate to **Interactive Sessions**.
* Click the button **Create New Session** and configure pipeline deployment.
* Follow the detailed guide here:
   [https://fptcloud.com/en/documents/model-testing-interactive-sessions/?doc=create-session](https://fptcloud.com/en/documents/model-testing-interactive-sessions/?doc=create-session)

---

### Step 3: Manage session

* The session list created successfully displays the **Interactive Sessions** page.
* In the **Actions** column, you can **run**, **cancel**, and **delete** sessions to manage effectively.
* Follow the detailed guide here:
   [https://fptcloud.com/en/documents/model-testing-interactive-sessions/?doc=manage-session](https://fptcloud.com/en/documents/model-testing-interactive-sessions/?doc=manage-session)

---

### Step 4: Evaluation model

* Open **Session list**.
* In the **Name** column, click a session you want to chat with the model.
* Open **Playground**, where you chat with the model to evaluate.
* Follow the detailed guide here:
   [https://fptcloud.com/en/documents/model-testing-interactive-sessions/?doc=evaluation-model](https://fptcloud.com/en/documents/model-testing-interactive-sessions/?doc=evaluation-model)

---

### Step 5: Integrate model

* Open **Session list**.1
* In the **Name** column, click a session you want to chat with the model.
* Open **Playground**.
* Click icon **Get API Endpoint** and copy the link.
* Follow the detailed guide here:
   [https://fptcloud.com/en/documents/model-testing-interactive-sessions/?doc=integrate-model](https://fptcloud.com/en/documents/model-testing-interactive-sessions/?doc=integrate-model)
